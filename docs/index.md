# Openbox Dotfiles

Welcome to my custom Openbox configuration! This repository contains a highly polished, lightweight, and plug-and-play setup for the Openbox stacking window manager. 

Openbox is an extremely fast, lightweight, and highly configurable window manager for the X Window System. Unlike tiling window managers, Openbox is a stacking window manager, meaning windows can overlap and be freely resized, but it still offers powerful keyboard-driven workflows and deep customization through its XML-based configuration files.

## Features
- **Lightweight & Fast**: Minimal resource footprint, perfect for older hardware or maximizing system performance.
- **Dual Conky Setup**: Beautiful system monitoring on both the left and right sides of the screen.
- **Auzia-Conky Integration**: Stunning Lua-based conky theme for the left panel.
- **Pre-configured Keybinds**: Carefully mapped keyboard shortcuts in `rc.xml` for a seamless workflow.
- **Custom Wallpapers**: A curated selection of Arch Linux-themed wallpapers included.
