# Theming

Openbox is themed using standard Openbox themes, which control window borders, title bars, and buttons.

## Configuration Files

- **`rc.xml`**: The main configuration file. Look for the `<theme>` section to change your Openbox theme.
- **`menu.xml`**: Defines the right-click desktop menu and application categories.
